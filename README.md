# GenCheck – AI Presentation Detection System
Overview
GenCheck is a web-based AI-powered system designed to detect whether PowerPoint (PPT) and PDF presentations are generated using AI tools or created manually by humans. The platform analyzes presentation content using Natural Language Processing (NLP), metadata inspection, and design pattern analysis to determine the likelihood of AI involvement.
GenCheck aims to promote transparency, academic integrity, and ethical use of AI in educational and professional environments.

# Problem Statement
With the rapid growth of AI presentation tools such as Gamma AI, Tome, ChatGPT Slides, and Beautiful.ai, entire presentations can now be generated in seconds. While these tools improve productivity, they also create challenges in maintaining authenticity, originality, and fairness in presentation submissions.
Currently, there is no reliable system that can automatically detect whether a presentation has been generated by AI or created by a human.

---
# Solution
GenCheck provides an intelligent system that analyzes uploaded presentations and identifies AI-generated patterns using multiple detection techniques.
The system evaluates text patterns, slide structures, metadata, and design similarities to determine whether the content is AI-generated or human-created. It also provides slide-wise insights and suggestions to improve originality.

# Key Features
1. AI-generated PPT and PDF detection
2. Natural Language Processing (NLP) analysis
3. Metadata inspection of documents
4. Design pattern recognition in slides
5. Slide-by-slide AI probability analysis
6. Suggestions to improve human-like originality

# How GenCheck Works
1. The user uploads a PowerPoint or PDF presentation.
2. The system extracts slide content, structure, and metadata.
3. AI detection models analyze the document using NLP and design pattern recognition.
4. The platform calculates the probability of AI-generated content.
5. A detailed report is generated with slide-level insights and suggestions.
